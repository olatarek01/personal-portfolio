## Overview
A simple and well-structured personal portfolio website built using pure HTML.  
The project highlights my professional profile, skills, experience, and includes a contact form in a clear and organized layout.

## Key Sections
- Professional header with name and title  
- About Me section  
- Skills list  
- Social media links  
- Experience timeline  
- Contact form  

## Technologies
- HTML5

## Getting Started
1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser.
3. No additional setup is required.

## Project Notes
- Built using HTML only without CSS frameworks or JavaScript.
- Styling is applied using basic internal and inline HTML.
- Focuses on structure, layout, and semantic HTML elements.
